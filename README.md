# Fellowship-EDA
Experiments for Fatima Fellowship 22

## Setup environment
```
pip install -qr requirements.txt
```
## Set pythonpath
```
# From main dir
export PYTHONPATH="$PWD"
```
## Config file
Change hyperparameters in `sample_configs/base_config.yaml`

## Training

```
python3 pipeline.py --conf `path to config file`
```
